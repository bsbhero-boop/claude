# 배움터 직무교육 통계 대시보드 — 사용 안내 (ver4)

노인맞춤돌봄서비스 종사자 직무교육의 **원데이터(엑셀)를 올리면 통계가 자동으로
계산**되는 도구입니다. 배움터 LMS 관리자 통계가 원활하지 않을 때, 다운로드한
원데이터로 이수율·미이수자·미응시자 등을 즉시 확인할 수 있습니다.

> **ver4 추가:** ① 이수율 현황표(배정·채용 인원 대비 이수율, **배정/채용 직접 입력**)
> ② 미응시·재응시 탭에 **차수 필터 + 이수자 자동 제외** ③ **중복자 확인** 탭
> ④ 데이터 현황(연도·차수) ⑤ 직군변경 보류 검토.

> **파일 하나**(`lms-statistics-v4.html`)면 끝입니다. 설치·인터넷 불필요.
> 모든 계산은 **내 PC 브라우저 안에서만** 이루어지며, 이름·ID 등 개인정보가
> 외부로 전송되지 않습니다.

---

## 1. 여는 방법

- **방법 A (가장 간단):** `lms-statistics-v4.html` 파일을 내려받아 **더블클릭** → 크롬/엣지에서 열림
- **방법 B (웹 주소):** GitHub Pages가 켜져 있으면
  `https://<사용자>.github.io/<저장소>/lms-statistics-v4.html` 로 접속

크롬 또는 엣지(Edge) 사용을 권장합니다.

## 2. 사용 순서

1. **① 회원정보** 영역에 회원정보 엑셀을 올립니다.
   `ID·사용자유형(직군)·교육구분(신규/경력)·교육대상여부` 등이 들어있는 파일입니다.
   → **한 번만** 올리면 브라우저에 저장되어, 다음에 열 때 자동으로 불러옵니다.
   (회원정보가 갱신되면 그때 다시 올리면 됩니다.)
2. **② 수강생목록** 영역에 `온라인통합수강생목록` 파일(들)을 올립니다.
   여러 파일을 한꺼번에 끌어다 놓아도 됩니다. → **올리는 즉시 자동 집계**됩니다.
3. 위쪽 탭에서 원하는 통계를 확인하고, 각 표의 **엑셀 다운로드** 버튼으로 내보냅니다.

> ⚠️ 정확한 이수율을 보려면 **해당 기간의 수강생목록 파일을 모두** 함께 올리세요.
> 일부만 올리면 그만큼 미이수자가 많게 나옵니다.

## 3. 탭 설명

| 탭 | 내용 |
|----|------|
| **요약** | 전체 대상자/이수자/이수율, **이수율 현황표**(직군별 신규/경력/소계 × 교육대상·배정(B)·채용(C) 대비 이수율, 배정/채용 직접 입력), 직군·경력별 현황 |
| **데이터 현황** | 업로드한 원데이터가 **연도·차수** 기준으로 어디까지 들어왔는지(최신 차수, 차수별 건수, 과정별 최신 차수, 교육신청일 범위) |
| **이수율 현황** | 시도 → 시군구 → 기관으로 펼쳐보는 이수율, 직군·경력별 이수율 |
| **이수자·미이수자 명단** | 대상자의 이수/미이수 현황(독려용 명단 겸용). **이수여부 필터**로 이수자 명단만 골라 다운로드 가능. 시도·직군·경력·검색 필터. 다운로드 파일에는 **시도·시군구·수행기관명·기관코드·직급**이 모두 포함됨 |
| **미응시·재응시** | **진도율 100% · 점수 0점**(강의는 끝냈으나 평가 미응시)인 사람. **차수 필터**로 해당 차수만 보고, **직무교육을 이미 이수한 사람은 재응시 대상에서 자동 제외**(이수여부 필터 제공) |
| **중복자 확인** | **하나의 ID가 같은 교육과정을 2회 이상 수료**한 건(같은 필수 과정을 여러 차수에서 중복 수료 등). ID는 본인인증 고유값이라 동명이인·중복계정 문제 없음 — 같은 과정 중복 이수만 점검 |
| **보류·직군변경** | 수료 **당시 직군**(수강목록)과 **현재 직군**(회원정보)이 달라 전직 가능성이 있는 건. 건별 **승인/반려** → 통계 즉시 반영 |
| **과목별 현황** | 과목별 신청자·수료자·수료율 |
| **개인 조회** | ID·이름으로 한 사람의 필수/선택 진행 상황과 이수 판정 확인 |
| **설정·도움말** | 이수 기준(차시) 변경, 데이터 관리 |

## 4. 이수(수료) 판정 기준

직무교육 계획서 기준입니다. **설정** 탭에서 값을 바꾸면 즉시 다시 계산됩니다.

- **대상 지역:** 회원정보의 시도가 **16개 시도**(서울·경기·인천·부산·대전·대구·울산·
  광주·강원·경남·경북·전남·전북·충남·충북·제주)인 경우만 집계합니다. **중앙·미상 등
  16개 외 지역은 모든 통계에서 제외**됩니다. (설정 탭에서 대상 시도 변경 가능)
- **대상자(모수):** 위 16개 시도 중 회원정보의 `교육대상여부 = Y` 이고 `상태 = 정상` 인
  **생활지원사·전담사회복지사**
- **신규자:** 해당 직군 **필수** 과정 수료 → 이수
- **경력자:** **필수** 과정 수료 **그리고** **선택** 과목 수료 차시 합계가
  기준 이상(생활지원사 **13차시**, 전담사회복지사 **10차시**) → 이수
- 원데이터의 `수료여부 = 수료` 를 그대로 사용하며, `수강취소` 건은 제외합니다.
- 같은 과정을 여러 차수에 수강했어도, **한 번이라도 수료**면 수료로 인정합니다.
- 전담사회복지사 과정은 **전담사회복지사·광역전담사회복지사**가, 생활지원사 과정은
  **생활지원사**가 이수 대상입니다(직군이 어긋난 수강은 이수로 인정되지 않음).
- **직군 변경(전직) 검토:** 수료 당시(수강목록)엔 생활지원사였다가 현재(회원정보)는
  전담사회복지사인 경우처럼 직군이 바뀐 건은 자동으로 **보류·직군변경** 탭에 모입니다.
  기본은 미이수로 두고, 검토 후 **승인**하면 그 사람의 당시 직군 수료를 인정해 이수로
  반영합니다(경력자는 선택 차시도 충족해야 함). 결정은 브라우저에 저장됩니다.
- 중간관리자·기타수행기관종사자 등 **이수기준이 정의되지 않은 직군**은 헤드라인
  이수율에서 제외되며, 요약 화면에 별도 인원으로 안내됩니다.

## 5. 자주 묻는 점

- **같은 파일을 실수로 중복 업로드하면?** **완전히 동일한 행은 자동 제거**됩니다.
  이수율·이수자 등 통계는 물론, 중복자 확인(같은 과정 중복 이수)도 영향을 받지 않습니다.
  제거된 중복 행 수는 상단 상태바와 데이터 현황에 표시됩니다. (원본 데이터 자체에
  들어있는 완전 중복 행도 함께 정리됩니다.)
- **느리지 않나요?** 수십만 줄도 처리합니다. 파일을 읽는 동안(수십 초) “처리 중”
  화면이 표시됩니다. 엑셀로 직접 수십만 줄을 다루는 것보다 빠릅니다.
- **데이터가 새어나가지 않나요?** 네트워크 전송이 전혀 없습니다. 브라우저 안에서만
  계산하고, 회원정보는 이 PC의 브라우저 저장소(IndexedDB)에만 보관됩니다.
- **파일 형식은?** `.xls`, `.xlsx`, `.csv` 모두 가능합니다. (배움터에서 받는 `.xls`
  그대로 사용 가능. 헤더 위에 제목/빈 줄이 있어도 자동으로 인식합니다.)
- **새 과목이 생겼어요.** 선택과목 차시는 **설정** 탭에서 추가/수정할 수 있습니다.
  매핑되지 않은 과목은 요약 화면에 알림으로 표시됩니다.
- **회원정보를 지우려면?** 설정·도움말 탭의 “저장된 회원정보 삭제”를 누르세요.
