# 파일명 고치기 (Mac Filename Fixer)

맥에서 만든 파일의 한글 이름이 다른 환경에서 `ㅍㅏㅇㅣㄹ.txt` 처럼 깨져 보이는
문제를, **드래그 앤 드롭** 한 번으로 고치는 오프라인 도구입니다.

## 왜 깨지나요?

macOS는 한글 파일명을 유니코드 **NFD(자모 분리형)** — `한` 을 `ㅎ+ㅏ+ㄴ` 으로
쪼갠 형태 — 로 저장합니다. 반면 윈도우·리눅스·대부분의 웹 서비스·압축 프로그램은
**NFC(완성형)** 를 기대하기 때문에, 맥에서 만든 이름이 다른 곳에서 깨져 보입니다.

이 도구는 파일·폴더 이름을 **NFC(완성형)** 으로 바꿔서 문제를 해결합니다.
(사본을 만들지 않고 **제자리에서 이름만** 바꾸며, 폴더는 통째로 재귀 처리합니다.)

## 어디서 고칠 건가요? — 플랫폼 선택

| 실행 환경 | 폴더 | 방식 |
|---|---|---|
| **윈도우**에서 고치기 (맥 파일을 윈도우로 받은 경우) | [`windows/`](windows/) | `FixNames.bat` 에 드래그 (PowerShell, 설치 불필요) |
| **맥**에서 고치기 (내보내기·업로드 전에 미리) | [`mac/`](mac/) | `build.sh` 로 만든 드롭 앱에 드래그 (AppleScript) |

> 지금 파일이 **윈도우에 있고 윈도우에서 고칠 거라면** → **[`windows/`](windows/) 폴더**를 보세요.

## 공통 특징

- **드래그 앤 드롭** 한 번 — 다른 기능 없음
- 사본 없이 **제자리에서 이름만** 변경 (내용·수정일 보존)
- **폴더 통째로** 처리 (하위 파일·폴더까지 재귀, 깊은 것부터 안전하게)
- **완전 오프라인** — 각 OS 기본 도구만 사용, 인터넷 불필요
- 이미 정상(NFC)인 파일은 건드리지 않고, 같은 이름이 있으면 덮어쓰지 않고 건너뜀

## 폴더 구성

```
filename-fixer/
├─ windows/   # 윈도우용 (FixNames.bat + FixNames.ps1)
├─ mac/       # 맥용 (build.sh → "파일명 고치기.app")
└─ README.md  # (이 파일)
```
