# 평가 문항 및 피드백 문구 예시 (산출물 7)

> 상황 판단(decision)·응급(emergency)·점검 퀴즈의 문항 설계 원칙과 예시.
> 실제 문항 전체는 `js/data.js`(`SCENARIOS`, `EMERGENCIES`, `QUIZ`)에 있으며 여기서는 대표 예시와 **피드백 문구 톤**을 규정합니다.

## 1. 피드백 작성 원칙
- **정답 피드백**: 짧게 확인 + 왜 옳은지 한 줄. 예) "⭕ 올바른 대응입니다! …(이유)…"
- **오답 피드백**: 비난하지 않기. "❌ 다시 생각해 볼까요?" + **왜 위험/부적절한지** + **올바른 대응** + **관련 수칙**.
- **위험 행동 오답**(의료행위·응급 오대응·개인정보 유출·금전 대납): 강하게 각인시키되 부드러운 톤. 재도전 유도.
- 문장은 존댓말·명령형 최소화, 한 문장 위주. 어려운 용어엔 쉬운 말 병기.

## 2. 상황 판단(decision) 예시 — 보기마다 이유 제공

**[모듈 2 · 개인정보]** 키오스크에서 주민번호 입력 중 뒷사람이 화면을 볼 수 있음.
| 보기 | 판정 | 피드백 |
|---|---|---|
| 시간이 걸리니 동행인력이 대신 빠르게 입력 | ❌ | 대신 입력은 자기결정 침해 소지. 어르신이 직접 하시도록 돕는 것이 먼저입니다. |
| **몸·손으로 화면을 가려 드리고 어르신이 직접 입력** | ⭕ | 노출을 막고 직접 입력하시도록 돕는 올바른 처리입니다. |
| 뒷사람에게 비키라고 큰 소리로 요구 | ❌ | 마찰을 유발합니다. 화면을 가려 조용히 노출을 차단하세요. |
> 관련 수칙: 개인정보 보호

**[모듈 6 · 금전]** "네 카드로 먼저 내고 나중에 주겠다."
- ⭕ 진료비는 어르신이 직접 결제하도록 돕고 영수증을 챙겨 드린다 → *금전 대납·대리사용 금지*.
- ❌ 편의상 동행인력 카드로 결제 → 분쟁 소지, 금지행위.

**[모듈 6 · 복약]** "약을 하루 두 번만 먹어도 되지?"
- ⭕ 임의로 바꾸면 안 된다고 말씀드리고, 약사에게 직접 확인하도록 돕는다 → *의료행위 금지*.
- ❌ "두 번만 드셔도 괜찮다"고 안내 → 복용량 판단은 의료행위.

## 3. 응급(emergency) 예시 — 2단계(대응 → 신고 대상)

**[모듈 8 · 낙상 / 병원 로비]**
- 1단계(초기 대응): **즉시 움직이지 않도록 안내 + 통증·출혈 확인** ⭕ / 바로 일으켜 세운다 ❌(2차 손상 위험)
- 2단계(신고 대상, **장소=병원 안**): **가까운 의료진·안내직원·보안요원 호출** ⭕ / 바로 119 ❌(병원 안에서는 의료진 우선)
- 공통: 보호자·기관 보고, **혼자 두지 않기**.

**[모듈 8 · 저혈당 / 귀가 이동 중]**
- 증상 인지: 식은땀·손떨림·피로·불안·두통
- 대응: 안전한 곳에 앉히고 **저혈당 응급식품**(주스 1/2컵·사탕 3~4개·설탕 1큰술 등) 섭취 도움 ⭕
  - 단, **의식이 없으면 억지로 먹이지 않기** ❌→ 기도 흡인 위험
- 신고(**장소=병원 밖·이동 중**): **119 신고 + 기관·보호자 보고** ⭕

**[모듈 8 · 경련 / 검사실 앞]** — 금지행위 구분이 핵심
- ⭕ 위험물 제거·안전공간 확보 / 조이는 것(넥타이·벨트) 풀기 / 옆으로 눕혀 기도 확보
- ❌ 팔다리를 붙잡아 억제 / 입에 손수건·손가락 넣기 (둘 다 위험 — 강한 각인 피드백)

## 4. 점검 퀴즈(QUIZ) 예시 — 8문항 중
1. 어르신이 "약이 많다"며 복용량을 줄여 안내했다 → **올바르지 않다**(의료행위).
2. 이웃에게 어르신 병명을 알려줬다 → **안 된다**(개인정보).
3. 키오스크 접수의 첫 단계는? → **본인확인**.
4. 수납 후 영수증은? → **챙겨서 어르신·보호자에게 전달**.
5. 진료실에서 동행인력 역할은? → **의사 설명 메모·질문 보조**(증상 설명은 본인).
6. 대기 중 가슴 통증 호소 → **즉시 가까운 의료진에게 알림**.
7. 서명이 어렵다는 어르신 → **직원에게 알리고 규정에 따름**(대리 서명 금지).
8. 동행 후 할 일이 아닌 것 → **진료 기록 SNS 공유**(절대 금지).
