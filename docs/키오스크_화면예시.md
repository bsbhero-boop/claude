# 키오스크 체험 화면 예시 (산출물 6)

> 무인 접수·수납 키오스크를 **어르신이 직접 조작하도록 보조**하는 능력을 기르는 실습.
> 아래는 화면 단위 텍스트 와이어프레임이며, 실제 동작은 `js/kiosk.js` + `js/data.js`(`KIOSK_FLOWS`)에 구현되어 있습니다.
> 각 화면 상단에는 항상 **동행인력용 팁**이 노출됩니다.

## 공통 UI
```
┌──────────────────────────────┐
│ 🖥️ 키오스크 실습          [✕] │
├──────────────────────────────┤
│ 💡 (동행인력 팁 — 이 단계에서 │  ← 노란 팁 바(항상 표시)
│    무엇을 어떻게 도울지)       │
├──────────────────────────────┤
│                              │
│      [ 화면 본문·큰 버튼 ]    │
│                              │
└──────────────────────────────┘
```

---

## 흐름 1 · 진료 접수 (register)

**화면 1-1 · 본인확인 방법 선택**
```
💡 팁: 어르신이 직접 누르시도록 옆에서 안내하세요. 신분증이 있으면 '신분증 투입'이 가장 간단합니다.

  본인확인 방법을 선택하세요
  ┌────────────────┐  ┌────────────────┐
  │ 🪪 신분증 투입  │  │ 🔢 주민번호 입력 │
  └────────────────┘  └────────────────┘
```
**화면 1-2 · (주민번호 선택 시) 키패드**
```
💡 팁: 개인정보가 주변에 노출되지 않도록 화면을 가려 드리고, 입력은 어르신 동의 하에 도와드립니다.

  주민등록번호 앞 6자리를 입력하세요
  [ • • • • • • ]
  ┌─┬─┬─┐
  │1│2│3│   ← 큰 키패드
  ├─┼─┼─┤
  │4│5│6│  …
```
**화면 1-3 · 진료과 선택** → 내과 / 정형외과 / 안과 / 신경과
**화면 1-4 · 접수 내용 확인** (환자명·진료과·예약시간·담당의) → `[접수하기]`
**화면 1-5 · 완료** → **대기번호 27번 · 본관 2층 내과 외래** / 팁: "대기번호표를 어르신께 드리고 함께 이동하세요."

---

## 흐름 2 · 진료비 수납 (payment)

- **2-1 본인확인** (신분증 / 대기번호표 바코드)
- **2-2 진료비 내역 확인** — 진찰료 15,800원 / 검사료 24,300원 / **본인부담금 12,030원(의료급여 적용)**
  - 팁: "금액을 천천히 읽어 드리고 의료급여·감면 적용을 함께 확인. 이상하면 원무과 문의."
- **2-3 결제수단** (신용/체크카드 · 현금)
  - 팁: "카드·현금은 어르신이 직접 넣으시도록. **동행인력이 금전을 받아 대신 결제하지 않습니다.**"
- **2-4 완료** — 영수증 1매 · 처방전 2매(약국 제출용·보관용) / 팁: "영수증은 귀가 후 보호자에게 꼭 전달."

---

## 흐름 3 · 증명서 발급 (certificate)

- **3-1 서류 선택** — 진료확인서 / 통원확인서 / 진료비 세부내역서
- **3-2 본인확인** (주민번호 6자리)
- **3-3 수수료 확인** — 진료확인서 1매 1,000원 / 팁: "결제는 어르신이 직접."
- **3-4 완료** — 서류 출력 / 팁: "이름·날짜 확인 후 구겨지지 않게 보관."

---

## 설계 포인트 (교육 의도)
1. **자기결정 존중** — 모든 흐름은 "동행인력이 대신"이 아니라 "어르신이 직접, 옆에서 보조"를 반복 각인.
2. **개인정보 보호** — 주민번호 입력 단계에서 화면 가림·노출 차단을 팁으로 강조.
3. **금전 원칙** — 결제 단계에서 대납 금지를 반복.
4. **실패 부담 없음** — 잘못 눌러도 되돌아가기 가능, 완료까지 자유 반복.
