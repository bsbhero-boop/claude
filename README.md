# 페르시아의 왕자 3D — 시간의 모래 🏰⏳

웹 브라우저에서 바로 즐길 수 있는 페르시아의 왕자 스타일 3D 플랫포머 게임입니다.
Three.js 기반의 단일 HTML 파일로, 빌드 과정 없이 실행됩니다.

## 실행 방법

```bash
# 저장소 폴더에서 간단한 정적 서버 실행
python3 -m http.server 8000
# 브라우저에서 http://localhost:8000 접속
```

또는 `index.html`을 브라우저로 직접 열어도 됩니다 (인터넷 연결 필요 — Three.js를 CDN에서 불러옵니다).

GitHub Pages를 켜면 (Settings → Pages → 브랜치 선택) 바로 온라인으로 플레이할 수 있습니다.

## 조작법

| 키 | 동작 |
|---|---|
| `W` `A` `S` `D` / 방향키 | 이동 |
| `Space` | 점프 |
| `Shift` | 질주 |
| `R` (길게) | ⏳ **시간 되감기** — 시간의 모래 게이지 소모 |
| `Enter` | 사망 시 체크포인트 부활 |

모바일에서는 화면의 가상 조이스틱과 점프 · ⏳ 버튼으로 플레이할 수 있습니다.

## 게임 요소

- **시간 되감기** — 함정에 빠지거나 추락해도 R키를 길게 누르면 최대 8초 전까지 시간을 되돌립니다. 죽은 직후에도 되감기로 부활할 수 있습니다.
- **시간의 모래** — 레벨 곳곳의 황금 조각을 모으면 되감기 게이지가 충전됩니다.
- **함정** — 솟아오르는 가시, 무너지는 다리, 회전 칼날을 피해 전진하세요.
- **체크포인트** — 황금 아치를 지나면 부활 지점이 갱신됩니다.
- **목표** — 궁전 가장 깊은 곳, 높이 솟은 황금 문에 도달하면 클리어!
