# 병원동행인력 교육 시뮬레이션 플랫폼

보건복지부 **퇴원환자 돌봄군 지원서비스**(노인맞춤돌봄서비스)의 **병원동행인력**을 위한
웹 기반 3D 교육 시뮬레이션 플랫폼입니다. 별도 설치 없이 브라우저(PC·모바일)에서 바로 동작합니다.

## 주요 기능

| 메뉴 | 내용 |
|---|---|
| 🏥 **병원 둘러보기** | 3D 가상 병원의 8개 장소(정문, 키오스크, 원무과, 대기실, 진료실, 검사실, 약국, 화장실)를 자유롭게 탐색하며 장소별 **해야 할 업무**와 **주의사항**을 학습 |
| 🚶 **진료 동행 절차 시뮬레이션** | 사전 준비·만남 → 접수 → 진료 대기 → 진료실 동행 → 검사 → 수납 → 약국 → 귀가·결과 전달까지 **9단계 표준 절차**를 캐릭터(동행인력 + 어르신)가 이동하며 단계별 체크리스트로 시연 |
| 🖥️ **키오스크 사용 실습** | 진료 접수 / 수납(진료비 결제) / 증명서 발급의 실제 키오스크 화면 흐름을 터치로 직접 연습. 각 단계마다 동행인력용 안내 팁 제공 |
| ✅ **핵심 수칙 · 학습 점검** | 의료행위 금지, 개인정보 보호, 금전 관리, 낙상 예방 등 8가지 핵심 수칙과 8문항 퀴즈 |

- **모바일 지원**: 터치 드래그로 3D 화면 회전·확대, 하단 시트형 학습 패널, 세이프에어리어 대응
- **학습 진행 저장**: 완료한 과정은 localStorage에 저장되어 메뉴에 ✓ 표시
- **3D 미지원 기기 대응**: WebGL을 사용할 수 없는 환경에서도 장소 버튼으로 모든 학습 내용 이용 가능

## 실행 방법

정적 사이트이므로 웹 서버로 폴더를 서빙하면 됩니다. (ES Module 사용으로 `file://` 직접 열기는 불가)

```bash
# 방법 1: Python
python3 -m http.server 8080

# 방법 2: Node
npx serve .
```

브라우저에서 `http://localhost:8080` 접속.

> 3D 렌더링은 Three.js(jsDelivr CDN)를 사용하므로 최초 로드 시 인터넷 연결이 필요합니다.

## 프로젝트 구조

```
index.html        # 시작화면(기관 로고)·메뉴·시뮬레이션·키오스크·학습 화면
css/style.css     # 모바일 우선 반응형 스타일
js/data.js        # 교육 콘텐츠 데이터 (장소별 업무·주의사항, 9단계 절차, 키오스크 흐름, 퀴즈)
js/hospital.js    # Three.js 3D 병원 씬 (건물·캐릭터·이동·카메라 추적)
js/kiosk.js       # 키오스크 터치 시뮬레이션
js/main.js        # 화면 전환·모드 제어·퀴즈 엔진
```

## 로고 교체 안내

시작화면의 **보건복지부 / 노인맞춤돌봄서비스 / 독거노인종합지원센터** 로고는
저작권 문제로 **임시 SVG 심볼**로 제작되어 있습니다.
실제 배포 시 `index.html`의 `.logo-row` 내 각 `<svg>`를 기관에서 제공받은
공식 로고 이미지(`<img src="...">`)로 교체하세요.

## 콘텐츠 수정

교육 내용(업무·주의사항·절차·퀴즈)은 모두 `js/data.js` 한 파일에서 관리합니다.
지자체·수행기관 지침에 맞게 문구를 수정하면 화면에 바로 반영됩니다.
