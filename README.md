# 병원동행인력 3D 체험 학습 플랫폼

보건복지부 노인맞춤돌봄서비스 **퇴원환자 돌봄군 지원서비스**의 **병원동행인력(돌봄제공인력)**을 위한
모바일 친화형 3D 체험 학습 플랫폼입니다. 별도 설치 없이 브라우저(모바일·PC)에서 바로 동작합니다.

『병원동행 서비스 교육 매뉴얼』을 기반으로, 실제 병원 동행 **하루 과정을 직접 판단하고 수행하며** 익히도록
**8개 모듈 '가상 병원동행 하루 체험'** 형식으로 구성했습니다. 화려한 3D보다 **절차 정확도·금지행위 인지·안전관리·응급대응**의 교육 효과를 최우선으로 설계했습니다.

## 가상 병원동행 하루 체험 (8개 모듈)

| 모듈 | 내용 |
|---|---|
| 1 🎒 **사전 준비** | 이용자 정보·일정·예약 확인, 준비물(신분증·진료카드·복용약·보조기구) 점검, 보호자 비상연락망 확인 |
| 2 🖥️ **병원 도착 및 접수** | 접수창구 찾기, **키오스크 보조 실습**, 본인확인·진료과·대기번호 확인, 휠체어 대여 판단, 개인정보 노출 차단 |
| 3 🪑 **진료대기** | 대기 안내·호출 확인, 화장실·휴게 동행, 장시간 대기 시 상태 관찰, 낙상 위험 예방 |
| 4 🩺 **진료 동행** | 진료실 이동 지원, 의사표현 보조, 안내사항 메모, 다음 절차 확인 (의료 판단·치료 결정 금지) |
| 5 🔬 **검사 이동 및 안내** | 검사실 위치, 금식 여부 확인, 검사 전후 어지럼·불안 확인, 안전한 이동 |
| 6 💊 **수납 및 약국 이용** | **키오스크 수납 실습**, 영수증·처방전 수령, 실손보험 서류, 약국 이동, 복약설명 청취 지원(임의 설명 금지) |
| 7 📝 **귀가 지원 및 기록** | 교통수단 확인, 안전 승하차, 상태 재확인, 보호자·기관 종료 전달, **서비스 제공 기록지 직접 작성** |
| 8 🆘 **응급상황 대응** | 낙상·의식저하/호흡곤란·저혈당·경련 초기 대응, **병원 안=의료진 호출 / 병원 밖=119** 장소별 신고, 혼자 두지 않기 |

각 모듈은 `3D 공간 진입 → 체크리스트 직접 수행 → "무엇을 해야 할까요?" 상황 판단 → 즉시 피드백`으로 진행되며,
**올바른 선택 시 다음 단계**, **잘못된 선택 시 왜 안 되는지·올바른 대응·관련 수칙**을 바로 안내합니다.

## 학습자(50대 이상) 친화 · 접근성 설계

- **🔈 읽어주기(TTS)**: 안내 문구·문항을 한국어 음성으로 낭독 (전역 토글, 브라우저 내장 기능)
- **🌱 연습 모드 / 📝 평가 모드**: 연습 모드는 틀려도 부담 없이 반복, 평가 모드는 이수 평가에 반영
- 큰 글자·큰 버튼, **한 화면 1과제**, 단순 메뉴, 탭(터치) 단일 조작 — 복잡한 게임 조작 없음
- 세로 화면 우선 반응형(넓은 화면은 우측 카드), 세이프에어리어 대응
- **중간 저장**: 완료 모듈은 localStorage에 저장되어 ✓ 표시, 언제든 이어하기·반복 학습

## 평가 및 이수 관리

- **📊 학습 결과 · 이수 현황**: 모듈별 점수 막대, **취약 항목**(정답률 낮거나 위험 행동 선택), **재학습 권장 모듈** 바로가기
- **이수 기준 점검**: ①8개 모듈 완료 ②상황 판단 정답률 70% ③응급 위험 행동 0건 ④점검 퀴즈 70%
- 위험 행동(의료행위·개인정보 유출·금전 대납·응급 오대응) 선택 시 **별도 경고 피드백**

## 자유 학습 · 복습

- 🏥 **병원 둘러보기**: 3D 병원 8개 장소를 자유 탐색하며 장소별 업무·주의사항 학습
- 🖥️ **키오스크 실습**: 접수 / 수납 / 증명서 발급 흐름을 터치로 연습
- 🧭 **실전 상황 대처**: 12가지 상황 판단 훈련
- ✅ **핵심 수칙 · 퀴즈**: 8대 수칙 + 8문항 점검

## 실행 방법

정적 사이트이므로 웹 서버로 폴더를 서빙하면 됩니다. (ES Module 사용으로 `file://` 직접 열기는 불가)

```bash
python3 -m http.server 8080     # 또는  npx serve .
```

브라우저에서 `http://localhost:8080` 접속.

## 프로젝트 구조

```
index.html        # 시작화면·메뉴(8모듈+복습)·시뮬레이션·키오스크·응급·기록·결과 화면
css/style.css     # 모바일 우선 반응형 스타일
js/data.js        # 교육 콘텐츠 데이터 (8개 모듈·응급 시나리오·장소·키오스크·퀴즈·기록지)  ← 콘텐츠 수정은 이 파일만
js/main.js        # 모듈 엔진·응급 엔진·결과 대시보드·TTS·모드·화면 전환
js/hospital.js    # Three.js 3D 병원 씬 (건물·가구·캐릭터 애니메이션·카메라 추적)
js/kiosk.js       # 키오스크 터치 시뮬레이션
assets/models/    # Kenney 3D 모델(GLB) — 가구 21종 + 애니메이션 캐릭터 (CC0/MIT)
vendor/           # three.js + OrbitControls + GLTFLoader (오프라인 동작용 로컬 번들)
docs/             # 서비스 기획안 및 산출물 (아래 참조)
```

## 폴백(저사양·미지원 기기 대응)

애셋 로드 실패 시 절차적(박스) 3D로, WebGL 미지원 기기에서는 장소 버튼·카드 목록으로 자동 전환되어 **3D 없이도 모든 학습 내용을 100% 이용**할 수 있습니다.

## 로고 교체 안내

시작화면의 **보건복지부 / 노인맞춤돌봄서비스 / 독거노인종합지원센터** 로고는 저작권 문제로 **임시 SVG 심볼**입니다.
실제 배포 시 `index.html`의 `.logo-row` 내 각 `<svg>`를 기관 공식 로고 이미지(`<img src="...">`)로 교체하세요.
